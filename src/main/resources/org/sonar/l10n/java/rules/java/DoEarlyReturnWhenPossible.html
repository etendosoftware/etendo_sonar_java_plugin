Conditional blocks of code should do an early return whenever possible. Early returns improve the code's performance and legibility

Noncompliant Code Example

  public boolean isPositive(int number) {
      if (number > 0) { // Noncompliant
        return true;
      } else {
        return false;
      }
  }

Compliant Solution

  public boolean isPositive(int number) {
      if (number <= 0) // Compliant
        return false;
      return true;
  }
